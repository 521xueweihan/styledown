Fancy style guide

Buttons

Add a class of .button to a <button> or <a> element, and use a size modifier (small, large, etc) and an appearance modifier (primary, success).

Normal button

Use the .button class.

<a class="button">Confirm</a><a class="button primary">Success</a>
ConfirmSuccess

Button sizes

Use the .button class.

<a class="button small">Confirm</a>
<br/><a class="button large">Success</a>
Confirm
Success

Simple table

Use .table on a table.

<table class="table">
  <tr>
    <th>One</th>

    <th>Three</th>
  </tr>
  <tr>
    <td>Hello there</td>

  </tr>
</table>
